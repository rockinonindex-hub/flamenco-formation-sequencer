# Flamenco Formation Sequencer

フラメンコ群舞のフォーメーションをコンパス単位で入力・再生する実証版です。

## 使い方

1. バイレ名と色を設定します。
2. パート名とコンパス数を設定し、「表を更新」を押します。
3. 表のセルを選択し、座標を直接入力するか、ステージ図をクリックして座標入力します。
4. `→` は移動中、空欄は前位置維持です。
5. `IN-L` / `IN-R` / `IN-B` は入場、`OUT-L` / `OUT-R` / `OUT-B` は捌け、`OFF` は舞台外です。
6. ステップ移動またはリアルタイム再生で確認します。

## 座標

- x=0 下手、x=4 上手
- y=0 奥、y=4 前
- 0.5刻み

## 保存

- ブラウザの localStorage に自動保存します。
- JSON保存・JSON読込にも対応しています。
- データはサーバーへ送信しません。

## 公開

静的ファイルのみで動作するため、GitHub Pagesで公開できます。
